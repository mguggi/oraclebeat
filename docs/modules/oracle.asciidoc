[float]
=== Example configuration

The Oracle Module module supports the standard configuration options that are described
in <<configuration-metricbeat>>. Here is an example configuration:

[source,yaml]
----
oraclebeat.modules:
- module: oracle
  metricsets: ["sesstats"]
  enabled: true
  period: 10s

  # List of Oracle connection strings
  hosts: ["oracle://system:S3cur3M3@localhost:1521/ORCLPDB1"]

  # Timeout after which time a metric-set should return an error
  # Timeout is by default defined as period, as a fetch of a metric-set
  # should never take longer then period, as otherwise calls can pile up.
  #timeout: 1s
----
